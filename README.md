# 生生護動 - 招生說明會成效分析 Dashboard

🎯 即時同步 Airtable Pipeline 的招生說明會成效追蹤系統

## 📊 功能特色

- ✨ **即時同步 Airtable**：每次刷新自動拉取最新 Pipeline 數據
- 📈 **自動計算 MQL**：MQL = SQL + 競品 + 合作夥伴 + 其他
- 🎯 **4 種動態圖表**：報名趨勢、轉換對比、園區分析、轉換率走勢
- 🔍 **園區篩選**：快速切換檢視特定園區
- 📱 **響應式設計**：支援桌機、平板、手機
